排除指南

一鍵自我排查三步驟

步驟 1：強制重新整理（清除快取）

請在網頁上按下組合鍵進行強制刷快取：

  • Windows / Chrome：按 Ctrl + F5
  • Mac / Safari：按 Cmd + Option + E 再重新整理

步驟 2：直接檢測 GitHub 是否發布成功

請直接點選下方任一路徑，如果顯示 404，代表 GitHub 尚未同步，或資料夾名稱實際在 GitHub 叫做中文的「圖片」：

測試 PPT 圖片連結 (images) 測試 Excel 圖片連結 (images) 測試 Word 圖片連結 (images)

步驟 3：中文字型檔名（若資料夾實際為「圖片」）

如果您的資料夾中文名稱確實是「圖片」，而非英文的「images」，則路徑需要改為中文格式。若經過測試確定是中文資料夾，請隨時通知我，我將立刻為您更換程式碼！
